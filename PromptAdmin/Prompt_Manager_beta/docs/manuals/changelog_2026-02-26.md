# Changelog - Día 1: 26 de Febrero, 2026 - El Nacimiento

Primer día de desarrollo intensivo. Creación de la estructura base y pivotaje tecnológico de la base de datos para asegurar persistencia.

### 🚀 Nuevas Funcionalidades
- **Arquitectura Base:** Lanzamiento de la Single Page Application con soporte para Casos de Uso (Prompts) y Reglas.
- **Gestión de Demos:** Módulo para enlaces a proyectos generados (Prosegur, Farmacity, Broker).
- **Consola de Debug:** Panel visual para monitorear el estado de las peticiones en tiempo real.
- **Portapapeles Contextual:** Botón para copiar automáticamente Reglas + Input para uso externo rápido.

### 🔧 Mejoras Técnicas & Backend
- **Pivotaje de BD:** Evolución constante para evitar bloqueos de CORS (KVDB -> Restful-api.dev -> JsonBlob -> **Firebase Firestore** como destino final).
- **Persistencia Firebase:** Implementación de guardado en la nube para persistencia global del usuario.
- **UI Responsiva:** Primeras tarjetas de Proyectos con soporte para rutas locales y URLs de GitHub.
