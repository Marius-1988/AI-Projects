# Changelog - Día 2: 27 de Febrero, 2026 - Conectividad & IA

Segundo día de evolución enfocado en la integración con Inteligencias Artificiales y refinamiento de la experiencia de usuario.

### 🚀 Nuevas Funcionalidades
- **Integración REAL con APIs de IA:** Conexión mediante `fetch` directo a Gemini, Claude y OpenAI.
- **Gestión Segura de Keys:** Almacenamiento local mediante `localStorage` (Privacidad absoluta).
- **Magic Zero Setup (Visor Web):** Detección automática de bloques HTML dentro de la consola para generar botones de previsualización en vivo (Blob URL).
- **Manual Model Override:** El usuario puede elegir entre Gemini, Claude y GPT.

### 🔧 Mejoras de UX & Bugfixes
- **Diseño Ultrawide de Modales:** Ajuste del ancho de los modales para permitir mayor espacio de edición y menos saturación de log.
- **Handshake Fallback (Gemini):** Implementación de la primera versión del selector automático de modelos ante fallos de cuota o falta de soporte para 1.5-flash.
- **Soporte de Edición:** Capacidad de modificar Casos de Uso ya guardados reutilizando el modal de creación.
